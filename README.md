# Sistema de Gerenciamento de Atendimentos

Projeto desenvolvido em Java com o objetivo de praticar Programação Orientada a Objetos (POO) por meio da implementação de um sistema de gerenciamento de atendimentos.

## Sobre o Projeto

A aplicação permite registrar e gerenciar atendimentos, aplicando conceitos fundamentais da linguagem Java e boas práticas de organização de código.

## Tecnologias Utilizadas

* Java
* Programação Orientada a Objetos (POO)

## Estrutura do Projeto

O projeto está organizado em pacotes para facilitar a manutenção e evolução da aplicação.

## Conceitos Praticados

* Classes e objetos
* Encapsulamento
* Herança e polimorfismo (se aplicável)
* Coleções
* Organização em camadas
* Boas práticas de desenvolvimento Java

## Como Executar

1. Clone o repositório.
2. Abra o projeto em uma IDE Java.
3. Execute a classe principal da aplicação.
